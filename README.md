# ✨ VibeCheck AI | Intelligent Mood Journal

VibeCheck AI is a modern, immersive web application that analyzes the sentiment of your thoughts in real-time. Built with a focus on **Glassmorphism UI** and **Fluid Animations**, it transforms your journaling experience into a visual journey.

## 🌟 Key Features
- **Real-time Sentiment Analysis:** Uses a custom-built logic to detect 'Happy', 'Reflective', and 'Intense' moods as you type.
- **Dynamic Theme Shifting:** The entire background and glowing elements shift colors smoothly based on your emotional vibe.
- **Immersive Glassmorphism UI:** Features a high-end design with blur effects, glowing orbs, and modern typography.
- **Persistent Storage:** Integrated with `LocalStorage` to ensure your journal entries are saved even after refreshing the page.
- **Smooth Animations:** Powered by `Anime.js` for hypnotic background movements and micro-interactions.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Advanced Flexbox & CSS Variables)
- **Library:** [Anime.js](https://animejs.com/) (for high-end animations)
- **Icons:** [Boxicons](https://boxicons.com/)
- **Logic:** Vanilla JavaScript (Sentiment logic & LocalStorage)
